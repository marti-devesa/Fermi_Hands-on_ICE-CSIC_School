# 9th Institute of Space Sciences Summer School: Fermi Hands-On

Fermipy tutorials for the summer school on Multimmessenger Astrophysics at ICE-CSIC (Cerdanyola del Vallès, Spain, July 2026). The following notebook and data will give a general introduction to fermipy for the analysis of Fermi-LAT data. This tutorial is inspired by those produced by the Fermip team [here](https://fermipy.readthedocs.io/en/latest/tutorials.html).
