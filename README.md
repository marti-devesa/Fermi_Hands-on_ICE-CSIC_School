# 9th Institute of Space Sciences Summer School: Fermi Hands-On

Fermipy tutorials for the [summer school on Multimessenger Astrophysics at ICE-CSIC](https://workshops.ice.csic.es/event/7/page/27-scientific-program) (Cerdanyola del Vallès, Spain, July 2026). The following notebook and data will give a general introduction to the analysis of Fermi-LAT data. This tutorial is inspired by those produced by the Fermipy team [here](https://fermipy.readthedocs.io/en/latest/tutorials.html).

*Hint*: In order to run the examples faster, you can download the setup output and the ROI already optimized from [here](
https://drive.google.com/file/d/1flLQbZ9P9VBhthFZJPJ1UeDhlCzWX6bj/view). These files are not necessary for the analysis itself.
